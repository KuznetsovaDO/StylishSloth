# StylishSloth — мобильное приложение для подписки на одежду
StylishSloth — это мобильное приложение для Android, которое позволяет пользователям оформлять и управлять подпиской на регулярную доставку одежды. Проект был разработан в рамках дипломной работы и представляет собой полноценное клиентское приложение с авторизацией, работой с базой данных и оплатой.



## Основные возможности
1. Регистрация и авторизация через Firebase Authentication
2. Выбор типа подписки (Базовая, Чёрная, Все включено)
3.  Указание размеров одежды и обуви
4. Управление адресами доставки
5. Личный кабинет с историей подписок
6. Редактирование профиля и подписок
7. Админ-панель через Firebase Firestore

## Технологии
* Язык - Java
* Платформа - Android
* Среда разработки - Android Studio
* База данных - Firebase Firestore
* Дизайн - Material Design 3

## Установка и запуск
1. Клонируйте репозиторий:
git clone https://github.com/KuznetsovaDO/StylishSloth
2. Откройте проект в Android Studio.
3. Настройте Firebase:
  Создайте проект в Firebase Console
  Добавьте файл google-services.json в папку app/
4. Запустите на эмуляторе или реальном устройстве.
